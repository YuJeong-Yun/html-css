padding 요소의 내부 여백

padding을 방향별로 각각 적용

  • padding-top:n; 위쪽에만 적용
  • padding-right:n; 오른쪽에만 적용
  • padding-bottom:n; 아래쪽에만 적용
  • padding-left:n; 왼쪽에만 적용

padding값의 갯수별 적용법

  • padding:n; 양사방 동일하게 적용
  • padding:n n; 상하 좌우가 서로 동일하게 적용
  • padding:n n n; 상,좌우가 서로 동일하게 적용,하의 순서
  • padding:n n n n; 상,우,하,좌의 순서
