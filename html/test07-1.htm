form-2

[ html5 form 요소]

1. input type="time" 시간 (오전/오후 시분)을 지정하는 요소

2. input type="date" 날짜 (연월일)를 지정하는 요소


min="1900-01-01" max="2200-12-31"로 최소 최대값 제한

3. input type="datetime-local" 시간과 날짜(연월일 오전/오후 시분)을 지정하는 요소

4. input type="week" 날짜(연주)를 지정하는 요소

5. input type="month" 날짜(연월)를 지정하는 요소

6. input type="number" 숫자값을 지정하는 요소


min="0" 로 최소값 제한
max="0" 로 최대값 제한
step="12" 로 증감 간격

7. input type="range" 범위내의 값을 지정하는 요소

8. input type="tel" 연락처(전화번호)를 지정하는 요소

모바일 브라우저에서 가상 키패드가 숫자 형식으로 바뀌고 전화 걸기 가능

9. input type="email" 연락처(이메일)를 지정하는 요소

브라우저에서 알맞은 형식인지 체크해 줌(@유무 등)
모바일 브라우저에서 가상 키패드가 영문 형식으로 바뀌고 @, .com등 나올 수 있음.

10. input type="url" 연락처(웹주소)를 지정하는 요소

브라우저에서 알맞은 형식인지 체크해 줌(http:// 유무 등)
모바일 브라우저에서 가상 키패드가 영문 형식으로 바뀌고 http://, .com등 나올 수 있음.

11. input type="color" 색상값을 지정하는 요소

[ input 속성들]

size="n" 상자의 길이

2. maxlength="n" 최대값 지정

3. minlength="n" 최소값 지정

4. placeholder="***" 입력예시나 설명 등을 지정


5. autocomplete="on/off" 자동완성 기능을 켬/끔

6. autofocus(aotofocus="autofocus") 자동 포커싱 지정

7. required (required="required") 필수입력 요소로 지정

8. readonly (readonly="readonly") 읽기만 가능한 요소로 지정

9. disabled (disabled="disabled") 사용 불가한 요소로 지정

전송시 함께 전달 불가

fieldset legend

폼 요소들을 그룹화하여 묶어서 표현. fieldset로 묶어주고 해당 그룹의 제목을 legend로 지정.
